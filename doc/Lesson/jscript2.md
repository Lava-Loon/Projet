() : utilisées pour :
 -Indiquer que c'est une fonction.
 -Ajouter des paramètres.

{} : définissent le corps de la fonction, où l'on écrit ce qu'elle fait.

const : Mot-clé JavaScript pour déclarer une variable constante.


document : Objet global de JavaScript représente toute la page HTML.

.createElement('div') : Méthode de l'objet document.
Sert à créer dynamiquement un élément HTML.

.classList : Propriété d'un élément HTML permettant de gérer ses classes CSS.

.add('X') : Méthode qui ajoute une classe CSS à l'élément X.

.style : Propriété qui permet de modifier les styles CSS d'un élément.

.left : Propriété CSS qui détermine la position horizontale de l'élément sur la page.

.width : définit la largeur de l'élément

.height : définit l'hauteur de l'élément

.apprendChild(X) : méthode JavaScript qui ajoute un élément HTML (X) en tant qu'enfant de Y

setTimeout : fonction qui execute une action après un certain délai

setInterval : fonction JavaScript qui execute une autre fonction à intervalles reguliers

.getElementById(X) : Méthode qui recupère un élément HTML avec l'Id="X"

.addEventListener('click', () => { ... }) : ajoute un écouteur d'évènements sur X. Quand on clique sur ce bouton,la fonction entre { ... } est exécutée

if : condition en JavaScript

=== : opérateur de comparaison stricte(vérifie aussi le type de donnée)

{ ... } : bloc de code exécuté si la condition est vraie
