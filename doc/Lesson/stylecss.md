margin : L'espace extérieur autour d'un élément, créant un espace entre cet élément et ceux qui l'entourent.

padding : L'espace intérieur entre le contenu de l'élément et ses bords.

box-sizing : Détermine comment les dimensions (width et height) d'un élément sont calculées.

font-family : Définit la police utilisée pour le texte de l'élément.

display : Contrôle le type de boîte de l'élément.

justify-content : Définit comment les éléments sont distribués horizontalement.

align-items : Définit l'alignement vertical des éléments dans une boîte Flex ou Grid.

min-height : Définit la hauteur minimale de l'élément.

background : Définit l'arrière-plan de l'élément.

background-size : Spécifie la taille de l'image d'arrière-plan.

background-position : Définit la position de l'image d'arrière-plan dans l'élément.

width : Définit la largeur de l'élément.

border : Définit les propriétés de la bordure de l'élément.

backdrop-filter : permet d’appliquer des effets graphiques à la zone derrière un élément.

box-shadow : Applique une ombre autour de l'élément.

color : Définit la couleur du texte.

border-radius : Définit les coins arrondis des bordures d'un élément.

font-size : Définit la taille de la police.

text-align : Définit l'alignement horizontal du texte à l'intérieur de l'élément.

position : Définit la manière dont l'élément est positionné dans le flux du document.

height : Définit la hauteur de l'élément.

outline : Ajoute une bordure autour d'un élément, en dehors de la bordure définie par border.

accent-color : Définit la couleur d'accent utilisée pour les éléments d'interface utilisateur natifs.

margin-right : Définit l'espace extérieur à droite de l'élément.

text-decoration : Ajoute des effets de décoration au texte.

cursor : Définit l'apparence du curseur lorsque la souris survole l'élément.

font-weight : Définit l'épaisseur de la police.

top : distance entre le haut de l'élément et le bord supérieur de son conteneur

left : distance entre le côté gauche de l'élément et le bord gauche de son conteneur

right : distance entre le côté droit de l'élément et le bord gauche de son conteneur

pointer-events : contrôle si un élément peut repondre aux clics ou survol de la souris

overflow : gère ce qui se passe quand le contenu dépasse la taille d'un élément

animation : permet d'animer des propriétés CSS . On definit les étapes et la durée de l'animation via une @keyframes
