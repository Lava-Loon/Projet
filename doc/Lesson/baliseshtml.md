<h1> : utilisée pour définir le titre principal d'une page

<div> : utilisée pour regrouper les elements

<input> : utilisée pour creer des champs de saisie dans un champ de formulaire

<label> : utilisée pour definir une etiquette pour un element de formulaire

<a> : utilisée pour creer des liens

<p> : represente un paragraphe de texte

<button> : utilisée pour creer un bouton

<form> : utilisée pour creer un formulaire

<canvas> : utilisée pour dessiner des graphiques, des animations ou d'autres types de contenus visuels de manière dynamique via JS
